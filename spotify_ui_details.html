Mobile UI Kit Resume

Project information

  • Category: UX/UI project
  • Project duration: FEB 2021 - APR 2021
  • Project URL: Mobile UI Kit Resume

I was inspired to create this project by a viral post I saw of somebody sharing their amazing Spotify-inspired resume. They said they made it for their dream job at Spotify. I was so impressed. It motivated me to create my own Spotify-inspired resume, which you can see in my portfolio too. I wanted to build on top of that and take it a step further by utlizing a model of Spotify's mobile app UI to act as a resume you can interact with. You'll see what I mean when you test it out yourself. :) The purpose of this project is to learn about design and research specific company fonts, colors and design best practices. The goal of this project is to practice my design and research skills as well as to help myself stand out for my current job search. Hope you enjoy, thanks for reading!

Tools used: Figma, Canva
